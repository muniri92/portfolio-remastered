SuccessApp

A single page web application that utilizes the IndeedJobsAPI and the EventbriteAPI to populate job applications and related corresponding local events that are aligned with the job search. Utilized HandlebarJS for dynamic HTML templating, PagesJS for client-side routing, and Heroku for deployment.

  • Link: SuccessApp
  • Github: Repo

Drone Quest

Drone Quest is an open source python module that allows a user to connect their Parrot AR-Drone and fly it using either our Angular web interface or a Leap Motion Controller. Used a Bottle framework to create an API that allows developers to install our module and control their Drone by hitting the correct API endpoints. As well as continuous integration using Tox, TravisCI and pytest.

  • Install Module: DroneQuest
  • Github: Repo

Equip

A web application that helps users create custom packing lists based on selected outdoor activities as well as a journal that allows users to document their trip. Built using vanilla JavaScript, HTML5, and CSS3. Utilized local storage to persistently keep the journal on the users current browser cache.

  • Site: Equip
  • Github: Repo

Mars Street View

Application that allows the user to navigate along the paths of the Mars rovers, while switching between each rover's multiple cameras. Developed a Postgres database on a Pyramid Framework, using the NASA Mars API to stores hundreds of thousands of photos and information about each photo respectively. Connected the database to front-end interface using JavaScript and jQuery. Successfully implemented pytest for both unit and functional testing as well as Tox to write code that was both Python 2 / Python 3 compatible.

  • Site: Mars-Street-View
  • Github: Repo
